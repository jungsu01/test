저장소에 있는 내용 내 PC에 다운로드
workgit 작업폴더 => 오마 => bash here
git clone 저장소주소
내 workgit 폴더에 저장소폴더 test 다운로드
cd test
(현재 작업중인 branch 확인하려면 git branch)
git add .
git commit -m "파일수정"
git push origin main

git commit만 하면 내 PC에만 저장되고, GitHub 저장소에는 아직 안 올라감.
GitHub까지 올리려면 마지막에 반드시:
git push origin main

dd → 업로드할 파일 선택
commit → 내 컴퓨터 Git 기록 저장
push → GitHub 저장소에 실제 업로드


[그냥 처음 저장소만 만들고 파일 업로드시]
git init
git remote add origin 저장소주소
git status
git add .
git commit -m "메시지"
git push origin main
